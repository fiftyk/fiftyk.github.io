箭头函数

① this 指向的固化，并不是因为箭头函数内部有绑定 this 的机制，实际原因是箭头函数根本没有自己的 this，导致内部的 this 就是外层代码块的 this。正因为它没有 this，所以也就不能用作构造函数。

② 箭头函数中不存在 arguments，super，new.target。
